Tworzy losową bazę danych z informacjami o uczniach, zawierającą:

## [Tankowania](tankowania.html) [[JS]](js/tankowania.js)

Tworzy losową bazę danych z informacjami o kolejnych tankowaniach samochodu, zawierającą:
- datę tankowania
- przejechany dystans
- ilość paliwa
- cenę paliwa
