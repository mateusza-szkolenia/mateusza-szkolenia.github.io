## Znaczniki

Znaczniki definiują elementy i ich zawartość

---

| znacznik  | element                |
|-----------|------------------------|
| `<p>`     | akapit (paragraf)      |
| `<img>`   | obraz                  |
| `<table>` | tabelka                |
| `<a>`     | odsyłacz               |
| ...       | ...                    |
